## About The Project

 <a href="https://github.com/anisharma07">

This is a dynamic speed typing website that allows users to create and participate in multiplayer typing matches. Users can challenge their friends in real-time typing competitions. The website provides a user-friendly interface where players can test their typing speed, accuracy, and improve their skills.

## Getting Started

Follow the steps belowðŸ‘‡ to get a local copy up and running...

### Prerequisites
Before running the projects download the following required softwares
* npm
  ```sh
  npm install npm@latest -g
  ```

* node.js
  ```sh
  download node.js from <https://nodejs.org>
  ```

* mongoDb
  ```sh
  download and configure mongoDb shell from <https://www.mongodb.com/try/download/shell>
  ```
Note: Alternatively the mongoDb URI can be generated by signing in to mongodb and creating a cluster on cloud.
### Installation

_Perform the following steps to run the project:_

1. Clone the repo
   ```sh
   git clone https://github.com/Teamexe/Type-Dash.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Create a .env file in projects root directory.

4. Type your mongoDb URI in .env file:
   ```sh
   MONGODB_URI=mongodb://127.0.0.1:27017/TypeDash
   ```  
- if you are using cloud based URI, then Use the following:

   ```sh 
   MONGODB_URI=mongodb+srv://username:password@cluster0.9lgdys4.mongodb.net/typeUserDB
   ```

4. Run the command:
   ```sh
   npm run dev
   ```
5. Start a type race at:
   ```sh
   http://localhost:3000
   ```
